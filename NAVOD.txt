INK RECIPE MANAGER — lokální verze (Printcolor / Pantone MS / katalog Stricker)
================================================================================

CO JE V BALÍČKU
  index.html            aplikace (otevírá se v prohlížeči)
  data.js               katalog 1 320 produktů Stricker (SCR/PDP/TXP/TRF/FIR)
  seznam_obrazku.json   seznam 5 583 obrázků ke stažení
  stahni_obrazky.py     stahovací skript (obrázky + knihovny pro offline běh)
  vzor_receptury.csv    vzorový formát pro hromadný import receptur

PRVNÍ SPUŠTĚNÍ (jednorázově, cca 5–15 minut)
  1. Rozbalte celý balíček do jedné složky (např. C:\InkManager)
  2. Nainstalujte Python 3, pokud ho nemáte: https://www.python.org/downloads/
     (při instalaci zaškrtněte "Add Python to PATH")
  3. Ve složce otevřete příkazový řádek a spusťte:
        python stahni_obrazky.py
     Skript stáhne fotky produktů a náhledy tiskových poloh do složky
     "obrazky" a knihovny do složky "lib". Lze ho kdykoli přerušit a
     spustit znovu — naváže tam, kde skončil.
  4. Otevřete index.html (dvojklik) — hotovo.

  Poznámka: s připojením k internetu funguje aplikace i bez kroku 3
  (knihovny si vezme z internetu), ale obrázky se zobrazí až po stažení.

POUŽITÍ
  KALKULACE   vyhledáte produkt (název/ref.), vyberete polohu (karty
              s náhledy jako na webu), recepturu a počet kusů — aplikace
              spočítá celkovou potřebu barvy v g/ml a rozpis komponent.
  PRODUKTY    katalog, ruční přidávání/úpravy, export CSV.
  RECEPTURY   Pantone standard i vlastní (Custom) receptury.
  IMPORT/DATA hromadné nahrání produktů (CSV/JSON) a receptur (CSV,
              formát viz vzor_receptury.csv), obnovení katalogu.

RECEPTURY PRINTCOLOR
  Pantone formule pro barvy Printcolor jsou licencovaná data — vyžádejte
  si je u svého obchodního zástupce Printcolor (recepturační software /
  formula guide k vašim řadám barev). Jakmile je dostanete, převedete je
  do formátu vzor_receptury.csv a nahrajete najednou v záložce
  Import / data → "Nahrát soubor receptur".

DATA A ZÁLOHA
  Úpravy se ukládají do prohlížeče (localStorage) na daném počítači.
  Zálohu katalogu pořídíte tlačítkem Export CSV v záložce Produkty.
  "Obnovit katalog z data.js" vrátí původní katalog Stricker.
